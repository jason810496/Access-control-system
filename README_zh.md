# IoT門禁系統

[English version](https://github.com/jason810496/Access-control-system/blob/main/README.md)

這是一個物聯網門禁系統的演示，功能包括讓業者在遠端、線上更改密碼和即時影像的功能。

## 演示

管理者登入界面

管理者頁面

- 更改密碼
- 確認當前密碼
- 清除特定資料
- 查詢資料
- 顯示使用者行為
- 加入固定運行模式（如每天傍晚6點自動更改密碼
